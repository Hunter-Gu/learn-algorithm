## 学习方式

不要死记硬背，不要为了学习而学习。

要学习它的“来历”、“自身的特点”、“适合解决的问题”以及“实际的应用场景”。

### 让你事半功倍的学习技巧

1. 边学边练，适度刷题。可以每个周末把当周的学习内容实现一遍。
2. 多问、多思考、多互动。
3. 知识需要沉淀，不要想试图一下子掌握所有。学习知识的过程是反复迭代、不断沉淀的过程。
