# 线性表 & 非线性表

## 线性表

线性表就是数据排成像一条线一样的结构，每个线性表上的数据最多只有前和后两个方向。

以下数据结构都是线性表结构：

- [数组](.array.md)
- [链表](./linked-list.md)
- [栈](./stack.md)
- [队列](./queue.md)

![线性表结构](@imgs/b6b71ec46935130dff5c4b62cf273477.jpg)

## 非线性表

非线性表与线性表相对立，之所以叫非线性，是因为数据之间并不是简单的前后关系。

以下数据结构都是非线性表结构：

- [树](../tree/README.md)
- [图](../graph/README.md)
